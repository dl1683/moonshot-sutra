# Archive

Historical documents from before the E2 reset. References to `artifacts/`,
`tools/`, old gates, old test counts, and old parameter estimates (180-250M)
reflect the state of the project at the time of writing, not the current build.

For current truth, see the files in `research/` (parent directory).
